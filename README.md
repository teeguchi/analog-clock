# アナログ時計
Canvas APIを利用したアナログ時計です。ほぼ、全画面で表示されます。

## 使い方
下部のボタンでダークモードに切り替わります。  
※画面サイズを変更した場合、リロードで時計のサイズを画面に合わせることができます。
